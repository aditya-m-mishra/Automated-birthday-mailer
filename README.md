Automated Birthday Mailer
Overview

Automated Birthday Mailer is a Python automation project that sends personalized birthday emails automatically using SMTP. The program reads user data from a CSV file, selects a random email template, customizes the message, and sends it to the recipient.

Features
Personalized birthday email automation
CSV-based user data handling
Dynamic email templates
SMTP email integration
Beginner-friendly Python project
Technologies Used
Python
Pandas
SMTP (smtplib)
Datetime Module
